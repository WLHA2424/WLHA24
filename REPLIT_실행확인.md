# Replit 봇 실행 확인 가이드

## 현재 상태 확인

### ✅ KeepAlive 웹서버
- Preview 패널에 JSON 응답이 보이면 KeepAlive 서버는 실행 중입니다
- 예: `{"status":"ok", "message":"봇이 실행 중입니다"}`

### ❓ 텔레그램 봇
- 터미널에서 다음 메시지가 보여야 합니다:
  - "KeepAlive 서버가 시작되었습니다"
  - "봇이 시작되었습니다"
  - "채널 메시지를 기다리는 중..."

## 봇 실행 방법

### 방법 1: 터미널에서 직접 실행
1. Replit 하단 터미널 클릭
2. 다음 명령어 입력:
```bash
python bot.py
```
3. Enter 키 누르기

### 방법 2: Replit Run 버튼
1. Replit 상단에 있는 초록색 "Run" 버튼 클릭
2. 자동으로 `python bot.py` 실행됨

## 실행 확인
터미널에 다음 메시지가 보이면 성공:
- ✅ "KeepAlive 서버가 시작되었습니다"
- ✅ "봇이 시작되었습니다"
- ✅ "채널 메시지를 기다리는 중..."
